JForum

JForum is an implementation of a basic web forum in Java Servlets/JSP. We developed it in a four-person team as the final project for Advanced Software Engineering (CPSC 410 UBC) for the fall 2008 term.

The system uses MySQL (no ORM), a Java backend (Servlets, JSP, JDBC), and HTML/CSS/JQuery on the frontend.

JForum is a basic forum with simple features, such as sections, topic threads, moderation, user authentication, and user profiles. JForum became an exercise in Java Servlet/JSP, HTML, CSS, and JavaScript, while simultaneously challenging the team to write best-practice code that handles requests in a straightforward MVC manner.

We developed under the waterfall model, including writing of specifications and user testing (grading).

Rareshare

This is a BitTorrent-like system written as the term project for a Computer Networking course at UBC. The system distributes a file across a peer-to-peer network, but uses Reed-Solomon coding to generate check segments that can act as wildcard segments, able to correct for any given segment of that size that is no longer available from any peer on the network.

Fall 2008

RareShare is a system designed to augment existing BitTorrent techniques in cases when sources for a piece of a file are scarce or missing. RareShare uses the Parity Volume Set format popularized for use with Usenet binary newsgroups. It was developed in a team of three as the term project for a distributed systems course (CPSC 416 UBC).

A sharing technique is designed such that each peer need only download any set of pieces whose size totals that of the original set of files to recover those original files. Best of all, this technique is backwards-compatible with conventional BitTorrent protocols.
